# my-ime2
> A Shuangpin IME for Chrome OS.  
> 用于Chrome OS的双拼输入法

## 项目说明
采用新技术构建的my-ime 2.0版本。相较于1.0版，新的my-ime2放弃了完全依赖在线输入api的做法，采用本地词库+用户输入自学习的方式构建一个完全离线可用的可扩展双拼输入法。

目前还处于开发阶段。

my-ime 1.0版：[Violin9906/my-ime](https://github.com/Violin9906/my-ime)
